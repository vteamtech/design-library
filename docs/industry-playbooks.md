## Style Families

Editions that naturally pair together for a coherent look. Pick one family as the base, then swap individual sections as needed.

## Quick Assembly Reference

### Fastest path to a live site (5-page site):

1. **Create 5 pages:** Home, About, Services, Contact, + one inner page
2. **Import templates** from the recipes above
3. **Global setup:** Import header + footer templates, set as theme templates
4. **Customize content:** Swap placeholder text/images with client content
5. **Style:** Adjust Global Site Settings (colors, fonts, buttons) — all templates inherit
6. **Verify:** Check at desktop + mobile + tablet
7. **Go live:** Change siteurl to production URL, regenerate CSS

### Per-section import workflow:
```
1. elementor-mcp-apply-template (import section to page)
2. Update text/images via elementor-mcp-update-widget
3. Verify with browser snapshot
4. Move to next section
```
